Overview

The Origin ecosystem builds from blockchain and IPFS data to provide the speed and features that enable real user growth and fast application development.

Software Systems

  • DApp - Example DAPP built on the Origin Protocol
  • Discovery Server - Fast access to Origin data over GraphQL
  • Notification Server - Live in browser notification of Origin events
  • Event Listener - Stream Origin events and data to other systems
  • Bridge Server - verifies identity attestations usings third parties
  • Messaging - Persistant storage of Origin messaging
  • IPFS Proxy - Anti-abuse IPFS proxy
  • Testnet Faucet - Handing out testnet OGN tokens on testnet
