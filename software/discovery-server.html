Discovery Server

The Origin discovery server provides fast access to Origin data using GraphQL. This allows dapps and other applications to show Origin data quickly.

Unified GraphQL API provides easy access to Listing, Offer and User objects that are stored either in PostgreSQL or elasticsearch. Api supports simple queries by property (e.g. ID) and when querying listings more advanced searches by searchQuery and searchFilters.

Running Locally

docker-compose up origin-discovery

The server should start and you can point your browser to http://localhost:4000 to access the GraphQL playground.
